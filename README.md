# Web Development Project 6 - *Quick Recipe Explorer*

Submitted by: **Shiyu Zhang**

This web app: **A pasta recipe dashboard powered by the Spoonacular API. Browse and filter 20 recipes by title, dietary preference, and cooking time, then explore interactive bar charts comparing cooking speed and calorie content across the dataset. Click any recipe row to navigate to a dedicated detail page — each with its own unique URL — showing a full ingredient list, nutritional breakdown, health score, price per serving, and more. Built with React, React Router for client-side navigation, and Recharts for data visualization.**

Time spent: **10** hours spent in total
